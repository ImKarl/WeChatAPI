# WeChatAPI

## 一、已支持功能清单

#### We.install

- 获取当前微信安装目录
- 获取安装的微信版本号
- 启动微信

#### We.login

- 获取登录的二维码内容
- 设置登录状态改变的监听器
- 获取当前登录状态
- 退出登录

#### We.message

- 发送文本消息
- 发送图片消息
- 发送文件消息
- 设置接收消息的监听器

#### We.user

- 获取登录账户信息
- 获取好友列表
- 同意添加好友

#### We.group

- 设置群公告
- 添加群成员

#### We.transfer

- 请求接收转账


## 二、使用步骤

- 安装对应版本的微信客户端
- 编译WeChatAPI，启动:cn.imkarl.wechat.AppLauncher
